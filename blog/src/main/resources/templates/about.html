About

The Tech

This project is just a demo application made to experiment with the Spring Boot-stack. It uses Spring Boot 2.1.9 in Java 11 (though no fancy features was used in the making of this project). It's a rather standard Server Side Rendered application using Thymeleaf, Spring Boot Data with MySQL, Spring Boot Security, Spring Boot Web and some other bits and pieces.

One noteworthy bit though, is FlexMark. It's a great project for converting between Markdown and HTML, with a lot of nice features. I did not want to spend my time on this project implementing an advanced editor with formatting, so I chose to throw up a simple textarea and allow the writers to use Markdown to format the articles. This is converted to HTML when the articles are rendered, and leaves the Markdown in the database for easy editing later.

Client side, I used Showdown to give the authors a live preview-area of the expected result though some minor differences, like background colors, will occur.

The Author

My name it Morten Olsrud. I've been enjoying the wonderful world of tech for as long as I can remember, and since I am a child of the eighties, that is as long time. Once I found out that I could make the computers do what I wanted, there was no turning back. I was hooked.

In 2003 I landed my first developer job, fresh out of school, with the self confidence only an ignorant n00b can have. Fast forward to today, and I'm still at it, though without the ignorance, a humbleness towards knowing all too well how much I still don't know.

During these years I've seen a lot more of SysOps than I want to admit, but I have still managed to do development in some form or other, creating internal tools or customer facing portals relevant to the ops-part. I've sunk my teeth in networking, security, servers and a lot of the Microsoft enterprise suit of software, like AD, DHCP, DNS etc.

I am familiar with a lot of things, even though I wouldn't call myself a Microsoft, networking, security or server expert. I am also not religious with regards to the tech and tools I use, though I do have my personal favorites, like we all do. Much love for VS Code, the Macbook and Docker Desktop! <3
