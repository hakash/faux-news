About the project

This is a demo project created by Morten Olsrud, imitating a typical blog or news site layout with flowing articles, notices and a large hero article at the top.

The project is made in Java with Spring Boot Framework using Thymeleaf for templating. It connects to a database, handles authentication and runs in the cloud. The project includes a simplistic editor for writing the articles as well, behind a login.

Check out the code on GitHub.
